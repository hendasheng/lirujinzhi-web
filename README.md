# Web AR 应用

## 文件大小
文件大小直接影响加载速度，尽可能缩减文件大小，保持顺畅浏览体验。

当前文件容量：
- 视频：3M/个
- 静态 3d 文件：757kb
- 动态 3d 文件：30M

动态文件（视频/3d）做成无限循环动画是不错的办法，既能缩减文件大小，还能假装一直在动。


## 文件格式
- 图像追踪
  - 视频 -> mp4
  - 3d -> gltf(含贴图) *当前未使用*

- ARKit
  - 3d -> USDZ
